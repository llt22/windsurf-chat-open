# WindsurfChat Open

🎉 免费开源 · 安全可控 · 无需配置

让 AI 停下来听你说话 - 随时介入对话，精准反馈指令

## 安装

1. 下载 `windsurf-chat-open-1.0.0.vsix`
2. Windsurf/VSCode: `Ctrl+Shift+P` → `Extensions: Install from VSIX...`
3. 选择下载的 .vsix 文件安装

## 使用

1. **安装后自动生效**，无需手动配置
2. **AI 工作时**，插件面板会自动弹出显示 AI 的请求
3. **输入反馈**：在文本框输入你的指令
4. **添加图片**：粘贴（Ctrl+V）或拖拽图片到文本框
5. **提交**：点击"提交"或按 `Ctrl+Enter`
6. **快速操作**：
   - 点击"继续" - AI 继续执行
   - 点击"结束" - 结束对话
   - 按 `Esc` - 快速结束

### 快捷键

- `Ctrl+Shift+W` - 打开插件面板
- `Ctrl+Enter` - 提交反馈
- `Esc` - 结束对话

## 特性

- ✅ 支持图片（粘贴/拖拽/预览/删除）
- ✅ 30 分钟超时保护
- ✅ 倒计时显示
- ✅ 浅色/深色主题自适应
- ✅ 全局配置，无需每个项目单独设置
